# AI Mastery Memory OS Pro

Local-first AI learning app with Prompt Lab, active recall and spaced repetition.

## Prompt Lab update

Prompt Lab now includes categorized prompting methods for:

- ChatGPT Core
- Research
- Codex / Coding
- Automation
- Learning
- Prompt Engineering
- Business
- RAG / Evaluation
- Decision Making
- Accuracy

Each prompt method includes best use case, why it works, a reusable syntax formula and a copy-ready template.

## Run locally

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```
